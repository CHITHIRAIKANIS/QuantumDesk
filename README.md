## Project Description

QuantumDesk is a real-time ticket management system where users can:
- Create tickets
- Send messages
- Update ticket status (Agent only)
- View real-time updates without refreshing

---

## Features

- Role-based access (Agent / User)
- Real-time chat using Socket.io
- Status management (Open, Pending, Resolved)
- JWT authentication
- Secure API routes

---

## How to Run

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Open:
   http://localhost:3000
